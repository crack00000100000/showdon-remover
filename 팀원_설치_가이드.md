# 쇼돈 자막 제거 도구 설치 가이드

## 1. 터미널 열기

`Cmd + Space` → "터미널" 입력 → Enter

## 2. 아래 한 줄 복사해서 붙여넣고 Enter

```
cd ~/Downloads && curl -fsSL -o install.command https://raw.githubusercontent.com/crack00000100000/showdon-vsr/main/install.command && chmod +x install.command && xattr -d com.apple.quarantine install.command 2>/dev/null && open install.command
```

## 3. 설치 진행

새 검은 창이 뜨면서 설치 자동 시작. 10~25분 소요.

도중에 두 번 액션 필요할 수 있음:
- Apple 다이얼로그가 뜨면 → "설치" 클릭
- `Password:` 가 뜨면 → Mac 로그인 비밀번호 입력 (화면에 안 찍혀도 정상) → Enter

## 4. 완료

`🎉 설치 모두 완료!` 메시지 뜨면 끝.
